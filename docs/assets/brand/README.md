# Codex Bridge Brand Assets

This directory contains the current repository branding pack derived from the provided PNG logo reference.

## Files

- `codex-bridge-logo-light.png`
- `codex-bridge-logo-dark.png`
- `codex-bridge-symbol-light.png`
- `codex-bridge-symbol-dark.png`
- `codex-bridge-avatar-light.png`
- `codex-bridge-avatar-dark.png`
- `codex-bridge-social-light.png`
- `codex-bridge-social-dark.png`
- `source/codex-bridge-logo-reference.png`

## Usage

- Use `codex-bridge-logo-light.png` for the main README hero image.
- Use `codex-bridge-logo-dark.png` when you need the full logo framed for dark surfaces.
- Use `codex-bridge-symbol-*` for tighter square crops without the wordmark.
- Use `codex-bridge-avatar-*` for avatar-like square placements with a bit more breathing room.
- Use `codex-bridge-social-*` for GitHub social preview or wider banner placements.

## Notes

- The current branding pack intentionally stays close to the provided source image.
- This pack is PNG-based. There is no SVG master in this iteration.
- The README hero is intentionally constrained to `width="420"` so the header remains balanced and not oversized.
